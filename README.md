# Primacura

<img src="assets/icons/app-logo.png" alt="Primacura Logo" width="200"/>

## Objective

Primacura is designed to enable doctors to:

- Add patients
- Write prescriptions
- Share prescriptions with patients
- Keep records of patients

## Current Features

- All patient and prescription records are stored locally on the device.

## Future Plans

- Move patient and prescription data to a remote database.
- Encrypt patients' personal information for enhanced security.

## Running the Flutter Code

To run the Flutter code for Primacura, follow these steps:

1. Ensure you have Flutter installed on your machine. If not, follow the instructions on the [Flutter installation guide](https://flutter.dev/docs/get-started/install).
2. Clone the repository:
   ```sh
   git clone https://github.com/mubashshirfirdausi/primacura.git
   ```
3. Navigate to the project directory:
   ```sh
   cd primacura
   ```
4. Get the required dependencies:
   ```sh
   flutter pub get
   ```
5. Run the application:
   ```sh
   flutter run
   ```

Make sure you have an emulator running or a physical device connected to your machine.

## Getting Started

To get started with Primacura, follow these steps:

1. Install the application on your device.
2. Register as a doctor.
3. Start adding patients and writing prescriptions.
